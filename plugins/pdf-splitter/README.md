# pdf-splitter

Split PDF files into individual pages as separate PDF or PNG files using TypeScript. Use when the user needs to extract pages from a PDF, convert PDF pages to images, or split a multi-page PDF into single-page files.

## Overview

node ~/.claude/skills/pdf-splitter/tools/dist/index.js <input.pdf>

## Installation

```bash
/plugin install pdf-splitter
```

## Usage

This plugin provides the `pdf-splitter` skill which Claude will use automatically based on context.
