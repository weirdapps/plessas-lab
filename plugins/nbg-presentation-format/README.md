# NBG Presentation Format

A skill for creating presentations and visual assets following National Bank of Greece (NBG) corporate brand guidelines.

## Features

- **Slide creation**: Cover slides, section dividers, content layouts, charts, tables
- **Color palette**: Complete NBG brand colors with usage guidance
- **Typography**: Aptos font family with proper sizing hierarchy
- **Icons**: SVG icon generation matching NBG design system
- **Charts**: Pre-configured chart colors and styling
- **Templates**: Ready-to-use slide templates for common use cases

## Usage

This skill is automatically triggered when you mention "NBG format", "NBG presentation", or "National Bank of Greece" in your requests.

## Dependencies

- PptxGenJS for presentation creation
- Sharp for image processing
- html2pptx workflow from document-skills

## Reference Files

- `references/icons.md` - Icon design specifications
- `references/colors.md` - Complete color palette reference
- `references/charts.md` - Chart styling and configuration
- `references/layouts.md` - Slide layout catalog
