# Claude Utility Plugins

Skills for productivity, file management, and external services

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add <your-github-username>/claude-plugins-utility
```

## Available Plugins

See the `plugins/` directory for available plugins.

## Plugin Installation

Install individual plugins:

```bash
/plugin install plugin-name@claude-plugins-utility
```
